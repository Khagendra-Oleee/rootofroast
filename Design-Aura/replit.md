# Roots of Roast - Coffee Portfolio Website

## Overview
A modern, visually stunning coffee portfolio website built with React, Vite, and Express. Features smooth animations with Framer Motion and an elegant dark theme.

## Project Structure
- **client/** - React frontend with components for the coffee portfolio
- **server/** - Express backend for serving the application
- **shared/** - Shared schemas and types

## Key Components
- **Hero** - Landing section with animated text
- **CoffeeTypes** - Interactive menu showcasing signature brews
- **Navigation** - Fixed top navigation bar
- **CoffeeShowcase** - Visual coffee display
- **Contact** - Contact form section

## Menu Items (Signature Brews)
1. Espresso - High intensity
2. Cappuccino - Medium intensity
3. Caffee Latte - Light intensity
4. Cold Brew - Medium intensity
5. Doppio - High intensity

## Recent Changes
- December 2024: Updated menu items - replaced "Pour Over" with "Caffee Latte" and "French Press" with "Doppio"

## Running the Project
```bash
cd Design-Aura && npm run dev
```

## Tech Stack
- React 19 with Vite 7
- Express.js backend
- Framer Motion for animations
- Tailwind CSS for styling
- TypeScript
