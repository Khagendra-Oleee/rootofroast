# Roots of Roast - Coffee Portfolio Website

## Overview
A modern, visually stunning coffee portfolio website built with React, Vite, and Express. Features sophisticated animations using Framer Motion, parallax effects, and an elegant dark theme.

## Project Structure
- **client/** - React frontend with components for the coffee portfolio
- **server/** - Express backend for serving the application
- **shared/** - Shared schemas and types

## Key Pages
- **Home (/)** - Landing page with hero, menu, case studies, and contact
- **Sourcing (/sourcing)** - Origin stories featuring Nepal, Ethiopia, and Colombia coffee
- **Space (/space)** - Urban sanctuary cafe culture and design philosophy
- **Craft (/craft)** - Espresso extraction study with technical details

## Menu Items (Signature Brews)
1. Espresso - High intensity
2. Cappuccino - Medium intensity
3. Caffee Latte - Light intensity
4. Cold Brew - Medium intensity
5. Doppio - High intensity

## Animation Features
- Smooth scroll reveal animations with stagger effects
- Parallax 3D layered motion on images
- Liquid morphing blob backgrounds
- Magnetic hover interactions on buttons
- Micro-interactions on cards (tilt, scale, color shifts)
- Smooth page transitions with blur effects

## Key Components
- **Hero** - Landing section with animated text
- **CoffeeTypes** - Interactive menu showcasing signature brews
- **CaseStudies** - Selected works with links to detail pages
- **Navigation** - Fixed top navigation bar
- **LiquidBackground** - Animated blob backgrounds
- **Magnetic** - Cursor gravity hover effect
- **TiltCard** - 3D tilt effect on hover
- **BlurText** - Text reveal animation
